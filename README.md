### Vector
A simple javaScript vector API.


## How to use

You can use [jsDeliver](https://www.jsdelivr.com/) to use this API.
Copy the following CDN to use

`https://cdn.jsdelivr.net/gh/jjimen30/Vector@latest/Vector.js`
