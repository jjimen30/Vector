### Vector
A simple javaScript vector API.


## How to use

You can take advantage of [jsDeliver](https://www.jsdelivr.com/) to use this API.
Simply copy the following CDN to use

`https://cdn.jsdelivr.net/gh/jjimen30/Vector@latest/Vector.js`
